Image Search Abstraction Layer


This App performs image search with Bing image search engine API

  • I can pass a string as a query parameter and I will receive the JSON response. Every search page contain 10 images metadata.
  • I can add the parameter ?offset=Page_number to paginate results.
  • I can view last 10 query history.

Example input:

Point your browser to the following addresses to seach "fuffy cat" images at page 3 (0 index based):

  • https://imgsrc-al.herokuapp.com/api/imagesearch/fuffy cat?offset=3

Point your browser to the following addresses to get last 10 query history:

  • https://imgsrc-al.herokuapp.com/api/latest/imagesearch
